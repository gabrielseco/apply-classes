# apply-clases

A small utility to apply classes, the use case to use it is in React.

# Instalation

```sh
  npm i --save apply-classes
  yarn add apply-classes
```

# Usage
```jsx
  const Example = () => (
    <div className={applyClasses({
      'example': true,
      'hidden': false
    })}></div>
  )
```

It will render only the truthy conditions
